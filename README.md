## 更新

2025.06.09

- 修复nameserves在开设双栈的虚拟机的时候设置格式有误的问题
- 修复上游带非全IPV6地址的(典型例子virtfusion开设的)宿主机导致无法分配IPV6独立地址的问题
- 添加ip6tables全端口映射的方法确保在极其受限的情况下依然可以给虚拟机配置独立的IPV6地址
